# Premier League Football Data Analysis

Data analysis projects using WhoScored data from the **2025/26 Premier League season**.

## Projects

- [Home vs Away Analysis](home_away_analysis/README.md) – home vs away performance comparison

## Data

- Raw data manually collected from [WhoScored.com](https://www.whoscored.com)
- Processed CSV files are stored in `data/processed/`
- Data dimensions: Overall / Home / Away, covering Summary, Defensive, Offensive, and xG categories

## Requirements

- Python 3.12+
- pandas
- matplotlib
- openpyxl

## Project Structure
